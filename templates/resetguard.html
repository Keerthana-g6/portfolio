Project Overview & Outcome

ResetGuard is a web-based security analysis tool designed to evaluate password reset mechanisms for vulnerabilities. It simulates real-world attack scenarios and provides actionable insights based on OWASP standards. The system successfully analyzes authentication endpoints and generates structured reports highlighting vulnerabilities and recommended fixes, improving overall security awareness and defensive design.

Technical Implementation & Features

Security Coverage Mapping:

[OWASP Top 10 (A01, A02, A05, A07)] → [Threat Classification: Low/Med/High/Crit] → [Risk Scoring System]

  • Core Tech Stack: Frontend built using React (Vite) and backend developed using Node.js and Express, utilizing LowDB for lightweight data persistence.
  • API & Integration: Designed a robust REST API for scan execution and result handling, featuring an integrated AI module for contextual security analysis.
  • Automated Detection: Automated vulnerability detection targeting rate limiting issues, brute-force risks, as well as token strength and expiry analysis.
  • Header Validation: Comprehensive security header validation (HSTS, CSP, etc.) combined with AI-generated remediation insights.

Future Improvements

  • Live Dashboard: Add a real-time attack simulation dashboard for dynamic monitoring.
  • External Integrations: Integrate external security APIs to enrich threat data and validation.
  • Production Scalability: Deploy the architecture with Docker for improved production scalability and environment consistency.
