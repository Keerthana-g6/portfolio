Project Overview

This project focuses on bridging hardware and software to create a compact, real-time ECG monitoring system. Standard clinical equipment is often bulky and expensive. This prototype was engineered to capture live cardiac signals and identify anomalies locally using lightweight, low-cost microcontrollers, bringing critical monitoring directly to the edge.

Technical Implementation

System Architecture Flow:

[Analog ECG Sensor] ➔ [ADC Module] ➔ [ESP32 Processing] ➔ [Python Prototyping] ➔ [C++ Implementation on ESP32] ➔ [Anomaly Output]

  • Hardware Integration: Interfaced an analog ECG sensor directly with an ESP32 microprocessor for continuous, real-time data acquisition.
  • Signal Processing:Prototyped signal filtering algorithms in Python, then optimized and implemented them in C++ on ESP32 for real-time execution with <20ms latency.
  • Anomaly Detection: Developed targeted logic to analyze waveform structure, successfully flagging common structural ECG changes and arrhythmias on the fly with ~92% estimated detection accuracy.
